python pip 설치시 ssl 에러 처리
------------------------------

### 기본
openssl의 실행 경로를 환경변수 `PATH`에 추가한다.

환경변수는 아래에서 변경가능함.

`설정 > 시스템 > 정보 > 고급 시스템 설정 > 환경 변수 > 사용자 변수 > PATH`


### 아나콘다 설치시
아나콘다내의 `C:\Users\<<user>>\anaconda3\Library\bin`를 `PATH`에 추가한다.
해당 경로에 openssl 실행파일을 찾을수 있다.
